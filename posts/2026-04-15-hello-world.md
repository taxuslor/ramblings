This is my first note. A place for random thoughts, code snippets, and things I want to remember.

## Why this exists

I wanted somewhere lightweight to jot things down — not a polished blog, just a scratchpad that happens to be public. Markdown in, HTML out, no fuss.

## How it works

```bash
./new.sh "Some thought I had"
# write markdown
./publish.sh "add new note"
```

That's it. The build script turns markdown into static HTML, GitHub Actions deploys it.

> The best note-taking system is the one you actually use.
